User Name and Password Checker

Username criteria:

  • Minimum 6 characters
  • Cannot contain $, #, or !
  • Cannot be the same as password

Password criteria:

  • Minimum 6 characters
  • Must contain either $, #, or !
  • Cannot be the same as password
  • Cannot be "password"
  • Must contain one upper and one lower case letter
  • Must contain a digit
